# 🧠 GitHub Profile Analyzer

Analyze any GitHub user's public profile to get:
- 📦 List of repositories
- 📈 Daily commit activity chart

Built with ⚡ **Vite + React + TypeScript**, styled using the modern and elegant ✨ **ShadCN UI**.
